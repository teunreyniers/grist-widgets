Setup Instructions

  1. Create an Availability table with:
    • A ShiftName text column for the shift title
    • A Status column (Choice type recommended)
    • Optional: additional columns for shift details
  2. Add this widget to the Availability table view
  3. In widget column mapping, select:
    • Shift Name ‚Üí your shift name/title column
    • Status ‚Üí your Status column
  4. Grant "Full document access" in widget settings
  5. Optional: Use "Select By" to filter records (e.g., by Assistant)
