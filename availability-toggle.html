Setup Instructions

  1. Create three tables: Shifts, Assistants, and Availability
  2. The Availability table should have:
    • A reference column to Shifts
    • A reference column to Assistants
    • A Status column (Choice type recommended)
  3. Add this widget and set "Select By" to the Assistants table
  4. In widget column mapping, select:
    • Shift Reference ‚Üí your Shift reference column
    • Assistant Reference ‚Üí your Assistant reference column
    • Status ‚Üí your Status column
    • Assistant Name ‚Üí the name column from Assistants
  5. Grant "Full document access" in widget settings
