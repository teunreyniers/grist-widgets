Setup Instructions

  1. Create three tables: Shifts, Assistants, and Availability
  2. The Availability table should have:
    • A reference column to Shifts
    • A reference column to Assistants
    • A Status column (text: "Available" or "Unavailable")
  3. Add this widget and configure columns in widget settings
  4. Use "Select By" to link to your Assistants table
  5. Grant "Full document access" in widget settings for write capability
