Setup Instructions

1. Create a table with columns for your filter config (e.g., FilterColumn, FilterValue)

2. In widget settings, map columns: select which column contains filter names and which contains values

3. Use "Select By" to link this widget to your config table

4. When you select/change rows in the config table, filters update automatically!
